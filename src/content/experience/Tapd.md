Founder of Tapd, building mobile software and supporting product development for [usetapd.com](https://usetapd.com) / [Tapd IG](https://www.instagram.com/usetapd/) / [Download the app!](https://apps.apple.com/us/app/tapd-focus-app-blocker/id6748239381).

Competed in Union College's SparkLab Program, where Tapd secured $20,000 in seed capital and won first place in the pitch competition among the eight teams that advanced through preliminary judging.

Independently developed an MVP iOS application for live demo in Swift, using Firebase for user authentication and profile management and NFC technology to support physical tag-based app blocking and device management.

Engineered a full-stack mobile application now on TestFlight using React Native and Node.js/Express, with RESTful APIs for complex data models, a PostgreSQL database hosted on AWS, AWS Cognito authentication and authorization, AWS Lambda serverless operations, and a CI/CD pipeline for testing and deployment.

Led user testing to improve first-time user experience and key app functionality based on feedback, and independently developed the website, landing pages, and ecommerce platform.

Worked to scale operation to universities and consumers mainly on the east coast.
